# ターミナルでファイルとリポジトリを作る

## インストールしたもの
- tig
  git でティグティグしやすくしてくれる
- hub
　github にハブハブしやすくしてくれる

## コマンド

リポジトリの作成
```sh
$ git init
```

空ファイル生成
```sh
$ touch READEME.md
```

vim でREADEME.mdを書く
```sh
$ vi READEME.md
```

vim のキー
- i INSERT 入力モード
- esc モード切替
- :wq vi 書いたファイルを上書き保存し、モード切替

git で状態を確認
```sh
tig status
```
